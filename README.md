# Responsive Ecommerce Website

## PayPal Checkout Integration

This website now includes a complete PayPal checkout integration with a lightweight backend. See [README-PAYPAL.md](./README-PAYPAL.md) for detailed setup instructions.

### Quick Start

1. Install server dependencies:
```bash
npm run install-server
```

2. Configure PayPal credentials in `server/.env` (copy from `server/.env.example`)

3. Start the backend:
```bash
npm run start-server
```

4. Open `index.html` in a browser or serve with:
```bash
python -m http.server 8000
```
